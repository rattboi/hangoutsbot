# MusicBot (based on HangoutsBot)

This is a bot made to interface with Google Hangouts. It uses the [`hangups`](https://github.com/tdryer/hangups) library to connect to hangouts. For more information about HangoutsBot, go to [`ovkulkarni/hangouts`](https://github.com/ovkulkarni/hangouts)

**NOTE: This project requires `python3`**
